OVERVIEW

웹사이트의 첫 이미지인 메인페이지의 부족한 컨텐츠양을 늘려 한눈에 사이트를 파악 할 수 있도록 구성하였으며, 기존 사이트의 컨텐츠 단순 나열로 인한 불편한 가독성을 개선하고자 하였습니다.
마우스의 움직임을 최소화 할 수 있는 스크립트를 적용하여 사이트 이용을 좀 더 효율적일 수 있게 하였으며,부족한 제품 소개 부분을 구체화하여 방문객을 구매사이트까지 유도하도록 개선하였습니다.

View Website

MAIN

DETAIL #1

swiper 플러그인을 이용하여 수평방향의 원페이지 스크롤 방식을 구현하였으며, 마우스 휠 뿐 아니라 키보드로도 페이지 이동이 이루어지도록 추가로 스크립트를 작성하였습니다.

SUB_강습안내

DETAIL #2

강습안내 첫 화면은 4개의 카테고리를 한 화면에 분할하여 사용자가 선택할 수 있게 하였으며, 선택 후에 해당 강습의 상세 내용이 보여지게 구성하였습니다.

SUB_예약 및 위치

DETAIL #3

달력과 form 태그를 이용한 예약 페이지를 작성하였으며, 옵션 선택 후에는 총 금액과 함께 선택된 옵션사항이 화면에 출력되어 사용자가 선택사항을 다시 확인 할 수 있게 하였습니다.
